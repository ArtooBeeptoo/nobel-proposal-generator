🚀 New Starts Guide

Guidelines and recommendations for new customer surgical starts.

✅ Minimum Requirements for New Starts

  • Surgical Kit - Appropriate PureSet™ kit for implant system
  • Prosthetic Kit - Basic, N1, or Full depending on scope
  • Implants - Initial inventory (minimum 10-20 units recommended)
  • Healing Abutments - Assortment of sizes
  • Cover Screws - Matching platform sizes
  • Regeneratives - If doing grafting procedures

📦 Recommended New Start Bundles

Basic Surgical Start

For general practitioners starting with implants

  • NobelActive® or NobelParallel™ CC PureSet™
  • Prosthetic PureSet™ Basic
  • 20 implants (assorted sizes)
  • Healing abutment assortment

Advanced Surgical Start

For experienced practitioners or specialists

  • NobelActive/NobelParallel CC Combo PureSet™
  • Prosthetic PureSet™ Full
  • 40 implants (full range)
  • Guided surgery option available

Digital Workflow Start

Full digital integration

  • Surgical kit + Guided surgery kit
  • TRIOS scanner or DEXIS iOS
  • X-Guide navigation (optional)
  • SprintRay printer (optional)
